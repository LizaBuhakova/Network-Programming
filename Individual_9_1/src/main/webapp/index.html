Servlet застосунок: Замовники та замовлення

Базовий контекст: /individual-9-1/customer

Доступні endpoints

  • GET /customer/all — список усіх замовників (JSON)
  • GET /customer/{id}/order/all — всі замовлення замовника (HTML або JSON через Accept: application/json)
  • GET /customer/{id}/order/{iid} — одне замовлення (JSON)
  • POST /customer/{id}/order/new — додати замовлення (JSON body)
  • PUT /customer/{id}/order/{iid}/edit — редагувати замовлення (JSON body)

Для HTML-представлення відкрийте: /individual-9-1/customer/1/order/all
